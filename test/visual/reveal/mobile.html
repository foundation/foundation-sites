Mobile (default)

By default the reveal sets scrollTop to 0 if getting opened on a mobile device (indentified via user agent) and restores the original scrollTop after being closed.

I'm a default reveal on a mobile device

Mobile (disabled mobile scrollTop)

This reveal uses has set the option `mobileScrollTop: false` and thus doesn't affect the scrollTop if opened.

I'm a reveal with the option mobileScrollTop set to false.
