Accordion Menu: Keyboard Control

  • Hitting the tab key focuses you to the menu or the last open item.
  • The up and down arrow keys should navigate you through the menu items.
  • The space bar, enter key and right arrow opens a menu if already closed.
  • The space bar, enter key, esc and left arrow opens a menu if already open.
  • Shift-tab and up arrow keys should navigate up the menu, including open sub-menus.
  • When focused on a link item, the space bar and enter key should navigate to the link.
