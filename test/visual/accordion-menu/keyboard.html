Accordion Menu: Keyboard Control

  • The Tab and down arrow keys should navigate down the menu, including open sub-menus.
  • Shift-tab and up arrow keys should navigate up the menu, including open sub-menus.
  • When focused on a parent menu, the space bar and Enter key should open or close the menu.
  • When focused on a link item, the space bar and Center key should navigate to the link.
