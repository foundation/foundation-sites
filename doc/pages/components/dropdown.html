{{> examples_dropdown_variables}} *** ## Semantic Markup With Sass You can create your own dropdowns using our Sass mixins.

Basic

You can use the `dropdown-container()` and `dropdown-style()` mixins to create your own dropdowns, like so: ##### The Container Mixin

SCSS

{{#markdown}} ```scss .custom-dropdown-container-class { @include dropdown-container(); } ``` {{/markdown}} ##### The List Style Mixin

SCSS

{{#markdown}} ```scss .custom-dropdown-container-class { @include dropdown-container(); li { @include dropdown-style; } } ``` {{/markdown}}

Advanced

You can further customize your dropdowns with the options in the `dropdown-container()` mixin:

SCSS

{{#markdown}} ```scss .custom-dropdown-container-class { @include dropdown-container( // Sets list-style. Default: list. Options: [list, content] $content:list, // Sets if dropdown has triangle. Default:true. $triangle:true, // Sets max-width. Default: $f-dropdown-max-width $max-width ); li { @include dropdown-style; } } ``` {{/markdown}} *** ## Configure With JavaScript It's easy to configure dropdowns using our provided JavaScript.
